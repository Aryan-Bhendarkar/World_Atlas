A modern web application that allows users to explore countries around the world, discover their details, and learn about their cultures.

## Features

- **Country Exploration**: Browse through a comprehensive list of countries
- **Search & Filter**: Find countries by name or filter by continent
- **Sorting Options**: Sort countries alphabetically (ascending/descending)
- **Detailed Information**: View detailed information about each country including:
  - Population
  - Region
  - Capital
  - Languages
  - Currencies
  - And more!

## Technologies Used

- React.js
- React Router
- CSS3
- REST Countries API
- Vite
